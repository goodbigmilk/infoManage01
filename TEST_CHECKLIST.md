# Windows 原生部署测试清单

## 测试前准备

- [ ] Windows 10/11 或 Windows Server
- [ ] 管理员权限
- [ ] 稳定网络连接
- [ ] 关闭杀毒软件（可选，避免误报）

## 基本功能测试

### 1. 安装测试

```powershell
# 以管理员身份运行
.\deploy-native.bat
```

**预期结果**:
- [ ] 成功下载 Go（显示进度）
- [ ] 成功下载 MySQL（显示进度）
- [ ] Go 安装完成（显示 Go 版本）
- [ ] MySQL 安装完成（显示 MySQL 版本）
- [ ] 应用编译成功（生成 infoManage.exe）
- [ ] 服务启动成功
- [ ] 显示访问地址: http://localhost:9901

### 2. 访问测试

```
在浏览器访问: http://localhost:9901
```

**预期结果**:
- [ ] 页面正常加载
- [ ] 没有 404 错误
- [ ] 可以看到主界面

### 3. 状态检查

```powershell
.\deploy-windows-native.ps1 status
```

**预期结果**:
- [ ] MySQL: 运行中（绿色）
- [ ] 应用: 运行中（绿色）
- [ ] 显示端口 9901
- [ ] 显示访问地址

### 4. 停止测试

```powershell
.\deploy-windows-native.ps1 stop
```

**预期结果**:
- [ ] 应用进程被终止
- [ ] 无法访问 http://localhost:9901
- [ ] MySQL 服务继续运行

### 5. 启动测试

```powershell
.\deploy-windows-native.ps1 start
```

**预期结果**:
- [ ] 应用重新启动
- [ ] 可以访问 http://localhost:9901
- [ ] 数据没有丢失

### 6. 重启测试

```powershell
.\deploy-windows-native.ps1 restart
```

**预期结果**:
- [ ] 应用先停止再启动
- [ ] 访问正常

## 高级功能测试

### 7. 自定义端口

```powershell
.\deploy-windows-native.ps1 stop
$env:PORT = "8080"
.\deploy-windows-native.ps1 start
```

**预期结果**:
- [ ] 应用在 8080 端口启动
- [ ] 可以访问 http://localhost:8080

### 8. 数据持久化

1. 创建一些测试数据
2. 运行 `.\deploy-windows-native.ps1 restart`
3. 检查数据是否还在

**预期结果**:
- [ ] 数据没有丢失
- [ ] 所有记录完整

### 9. 卸载测试（最后执行）

```powershell
.\deploy-windows-native.ps1 uninstall
```

输入 `UNINSTALL` 确认

**预期结果**:
- [ ] 应用停止
- [ ] MySQL 服务被删除
- [ ] C:\infoManage 目录被删除
- [ ] infoManage.exe 被删除
- [ ] 计划任务被删除
